## 🧭 Overview

This is my personal developer portfolio — a fully hand-crafted, multi-page website built from scratch without any CSS frameworks or UI libraries. It showcases my education, skills, projects, certifications, and a way to get in touch with me.

The design follows a **dark glassmorphism** aesthetic with gradient accents, animated blobs, scroll-reveal animations, a custom cursor, and a responsive layout that works seamlessly on all devices.

## ✨ Features

- **Dark Glassmorphism Design** — frosted-glass cards, layered depth, animated gradient blobs
- **Custom Cursor** — smooth trailing ring cursor effect
- **Scroll-Reveal Animations** — elements animate in as you scroll
- **Staggered Card Animations** — skills and cards animate in one by one
- **Animated Skill Bars** — proficiency bars animate on scroll into view
- **Project Filter Tabs** — filter projects by category (Web, Full-Stack, Design)
- **Certificate Preview Modal** — click to view certificate in a full-screen modal
- **Resume Download** — one-click PDF resume download
- **Fully Responsive** — mobile-first layout with hamburger navigation
- **Page Transition Overlay** — smooth fade between pages
- **Back to Top Button** — scroll-aware floating button
- **SEO Ready** — semantic HTML5, meta descriptions, proper heading hierarchy

## 🚀 Getting Started

No build tools or installations required. Just open the project locally:

### Option 1 — Direct Open
1. Clone or download the repository
2. Open `index.html` in your browser

```bash
# Clone the repository
git clone https://github.com/jalgandhi-dev/portfolio.git

# Navigate into the folder
cd portfolio

# Open in browser (Windows)
start index.html

# Open in browser (macOS/Linux)
open index.html
```

### Option 2 — VS Code Live Server (Recommended)
1. Install the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension in VS Code
2. Right-click `index.html` → **Open with Live Server**
